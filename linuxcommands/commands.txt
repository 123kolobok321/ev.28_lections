linux commands
Ctrl + Alt + T - команда окрытия терминала
~  символ домашней директории 
/ - символ корневой лиректории
. - обозначает текущую директорию
.. - обозначает предыдущую директорию
1. Ls - команда для вывода всех директорий и файлов в текущей(рабочей) директории в терминале
2. cd (change directory) - команда для перехода из одной директории в другую
3. mkdir - создаёт директории
4. touch -создаёт файлы
5. rmdir - удоляет пустую директорию
6 . rm - удоляет файл
7. rm -rf - удоляет вообще всё( принудительное удоление)
8.ls -a - показывает скрытые и обычные файлы и директории
ls -la - показывает подробную информацию
9.pwd (print working directory)- выводит путь до текущей директории
10. cp - копирует файл или директорию в указаное место
    cp <название> <куда>
11 mv - команда для перемещения файла или директории а так же для переименования файла или деректории
           переименование: mv john.txt jamie.txt
           перемещение: mv john.txt directory/
12. cat - показывает содержимое файла
13. nano - открывает файл для редоктирования в графическом интерфейсе нано *если нет файла то нано создаст новый
14. echo - команда для бысрой записи текста в файл
      echo текст > файл
